# Шпаргалка по работе с Git

---

## Список основых комманд:

### Навигация  
- `ls` - показать содержимое директории;
- `ls -a` - показать содержимое директории включая скрытые папки и файлы;
- `pwd` - показать, в какой я директории сейчас;
- `cd` - сменить директорию;
- `cd ..` - сменить директорию на __родительскую__ (на уровень выше);
- `cd ~` - сменить директорию на __домашнюю__;
- `cd /` - сменить директорию на __корневую__.

### Создание  
- `touch` - создать файл;
- `mkdir`- создать директорию;

### Копирование и перемещение  
- `cp` - скопировать файл или директорию (что и куда);
- `mv` - переместить файл или директорию (что и куда);

### Чтение  
- `cat` -распечатать содержимое текстового файла;

### Удаление  
- `rm` - удалить файл;
- `rmdir` - удалить директорию;
- `rm -r` - удалить директорию со всеми файлами в ней;

### Инициализация репозитория  
- `git init` - инициализируй репозиторий;

### Подготовка файл к коммиту  
- `git add` - подготовить файл к коммиту;
- `git add --all` - подготовить все файлы, в которых были изменения, и все новые файлы к коммиту;
- `git add .` -  подготовить текущую папку и все файлы в ней к коммиту;

### Создание коммита  
- `git commit -m "Коммент"` - создание коммита с комментарием;

### Просмотр информации о коммитах  
- `git log` - вывести подробную историю коммита;

### Просмотр состояния файлов  
- `git status` - вывести текущее состояние репозитория;
